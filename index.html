Bachelor Thesis: Nutrition App

One-page presentation of a Flutter mobile application prototype with local SQLite database, macro (calorie) calculator and vitamin calculators.

UA: Мобільний застосунок для розрахунку КБЖУ та вітамінів на базі Flutter і SQLite

EN: Mobile App for Macro & Vitamin Calculation based on Flutter and SQLite

Short description (≤300 chars): Flutter mobile nutrition app prototype with user account, SQLite storage, macro calculator and vitamin calculators (single and daily modes). Includes diet plan placeholder module.

Keywords: Flutter, Dart, SQLite, nutrition, calories, vitamins, diet plan, mobile application

Relevance

Nutrition tracking requires accurate calculations and a convenient interface. A mobile application allows users to quickly estimate energy and nutrient intake and analyze vitamin content based on food portions.

Goal

Develop a prototype of a mobile nutrition application using Flutter with local SQLite storage, authentication, macro (Kcal/P/F/C) calculation and vitamin calculation modules.

Main tasks (3–5)

  • Design and implement SQLite database schema and data access layer (DAO).
  • Implement authentication (register/login) with password hashing (salt + SHA-256).
  • Develop macro calculator: search foods, add portions, calculate total Kcal/P/F/C.
  • Develop vitamin calculators: single product mode and daily mode.
  • Create user cabinet (profile data and edit actions) and prepare a diet plan module placeholder.

Methodology

The application uses a layered structure (models/DAO/services/UI). Calculations operate on portion input (grams) using per-100g reference values stored in SQLite. Vitamins are retrieved using a relation table and JOIN queries.

Expected results

  • Working offline-first Flutter prototype with SQLite database.
  • Macro calculation for multiple foods and portions.
  • Vitamin calculation for a single product and daily list with share percentage.
  • Public repository with clean history and documentation.
